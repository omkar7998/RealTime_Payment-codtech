# RealTime_Payment-codtech

This project demonstrates how to integrate the Razorpay Payment Gateway into a **.NET Web API** with a **simple HTML frontend**.  
The project includes:

- Razorpay Order Creation (Backend)
- Simple & Stylish Frontend Payment Page
- Redirection to Success Page after Payment
- Clean UI with CSS
- Works perfectly in **Visual Studio Code**
